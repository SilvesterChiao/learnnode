## 常用 NPM 包

-   chalk: 终端输出着色工具
-   commander: 完整的 nodejs 命令行解决方案
-   globby: 用户友好全局匹配工具
-   inquirer: 交互式命令行界面
-   jscodeshift: AST 解析工具
-   colors: console 着色工具
-   address: ip 地址解析工具
-   opn: 打开浏览器
-   open: 打开浏览器
-   http-proxy-middleware: http 中间件
-   webpack-dev-middleware: webapck 开发环境中间件
-   webpack-hot-middleware: webpack 热更新
-   http-proxy: http 代理工具
